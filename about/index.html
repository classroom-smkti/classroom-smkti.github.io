cara ikut berkontribusi :
Daftar dahulu ke akun GitHub.com.
Lakukan Install dan konfigurasi Git. Setelah sukses melakukan install dan konfigurasi, fork classroom-smkti.github.io.
Buka git bash kemudian lakukan beberapa perintah berikut :
$ git clone https://github.com/classroom-smkti/classroom-smkti.github.io
$ mkdir /ngoprek/judul-oprek-yang-akan-di-postkan
$ cp /ngoprek/index.html /ngoprek/judul-oprek-yang-akan-di-postkan/
$ sublime /ngoprek/judul-oprek-yang-akan-di-postkan/index.html
  // Edit sesuai artikel yang di bahas
$ git pull
$ git push -u origin master
Berikutnya adalah melakukan pull request, tunggu kami melakukan merge dan postingan kamu telah dipublikasi.

Jika teman - teman masih belum bisa memahami cara ikut berkontribusi, bisa contack kita.
